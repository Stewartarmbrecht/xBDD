xBDD Defining Features

Initial Setup
	Generating a new project
		Generating an MSTest Project [Untested]
			Given you have the latest version of dotnet core installed
				See [Installing Dotnet Core]() for instructions. 
			And you open bash, powershell, or a command window
			And you Install the dotnet-xbdd tools:
				dotnet tool install --global dotnet-xbdd --version 0.0.4-alpha
			And you create a folder for your test project and navigate to the folder
			Create a new test project using the installed xbdd tools:
				dotnet xbdd init
			Build and run the new test project
				dotnet test
